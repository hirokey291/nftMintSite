 ## nftMintSite

ポートフォリオ用に作りました。

### はじめに
    このアプリは、ブロックチェーンとスマートコントラクトを使用して構築した、  
    NFTをmintするためのアプリケーション(DApps)です。

 ** アプリはこちらのURLから見れます！ &rarr; **&emsp; https://nft-mint-site-xk2d.vercel.app

### 開発の目的
    このアプリを開発した目的は、実際に手を動かしてDApps(分散型アプリケーション)を開発することで、  
    ブロックチェーン技術やDAppsの仕組みを理解し、ブロックチェーン、スマートコントラクトを実装する  
    実践的なスキルを身につけることにあります。
